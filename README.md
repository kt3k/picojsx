# @kt3k/picojsx v0.1.1

A minimal jsx runtime implementation. Renders jsx into a string.

Set the below in `tsconfig.json` or `deno.json`:

```json
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "@kt3k/picojsx@0.1.1"
  }
}
```
