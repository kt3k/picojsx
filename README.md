# @kt3k/picojsx

A minimal jsx runtime implementation. It renders jsx into a string.

Set the below in `tsconfig.json` or `deno.json`:

```json
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "@kt3k/picojsx"
  }
}
```
